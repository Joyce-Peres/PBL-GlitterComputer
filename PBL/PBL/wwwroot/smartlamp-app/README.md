# Purpurine Smart Lamp (PWA)

Arquivos do PWA original foram trazidos para dentro do PBL para facilitar abrir/rodar tudo pela Solution.

- App: `wwwroot/smartlamp-app/index.html`
- Manifest: `wwwroot/smartlamp-app/manifest.json`
- Service Worker: `wwwroot/smartlamp-app/sw.js`

Acesse enquanto o PBL estiver rodando em `http://localhost:5000/smartlamp-app/index.html` (ou a porta que o Visual Studio indicar).
